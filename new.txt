1. Passengers Table
Stores details about passengers.

Column Name	Data Type	Constraints	Description
PassengerID	INT	PRIMARY KEY	Unique identifier for passengers
FirstName	VARCHAR(50)	NOT NULL	Passenger's first name
LastName	VARCHAR(50)	NOT NULL	Passenger's last name
Email	VARCHAR(100)	UNIQUE, NOT NULL	Passenger's email address
PhoneNumber	VARCHAR(15)		Passenger's contact number
DateOfBirth	DATE		Passenger's date of birth
FrequentFlyerID	VARCHAR(20)	UNIQUE	Frequent flyer program ID

2. Flights Table
Stores details about flights.

Column Name	Data Type	Constraints	Description
FlightID	INT	PRIMARY KEY	Unique identifier for flights
FlightNumber	VARCHAR(10)	UNIQUE, NOT NULL	Flight number
DepartureAirport	VARCHAR(50)	NOT NULL	Airport code for departure location
ArrivalAirport	VARCHAR(50)	NOT NULL	Airport code for arrival location
DepartureTime	DATETIME	NOT NULL	Scheduled departure time
ArrivalTime	DATETIME	NOT NULL	Scheduled arrival time
AircraftID	INT	FOREIGN KEY	References Aircraft(AircraftID)

3. Bookings Table
Tracks passenger bookings.

Column Name	Data Type	Constraints	Description
BookingID	INT	PRIMARY KEY	Unique identifier for bookings
PassengerID	INT	FOREIGN KEY	References Passengers(PassengerID)
FlightID	INT	FOREIGN KEY	References Flights(FlightID)
BookingDate	DATETIME	NOT NULL	Date and time of booking
SeatNumber	VARCHAR(5)		Assigned seat number
BookingStatus	ENUM('Confirmed', 'Cancelled', 'Pending')	NOT NULL	Status of the booking

4. Crew Table
Manages crew information.

Column Name	Data Type	Constraints	Description
CrewID	INT	PRIMARY KEY	Unique identifier for crew members
FirstName	VARCHAR(50)	NOT NULL	Crew member's first name
LastName	VARCHAR(50)	NOT NULL	Crew member's last name
Role	ENUM('Pilot', 'Co-Pilot', 'Flight Attendant', 'Engineer')	NOT NULL	Crew role
ContactNumber	VARCHAR(15)		Contact number
AssignedFlightID	INT	FOREIGN KEY	References Flights(FlightID)

5. Aircraft Table
Stores information about aircraft.

Column Name	Data Type	Constraints	Description
AircraftID	INT	PRIMARY KEY	Unique identifier for aircraft
AircraftModel	VARCHAR(50)	NOT NULL	Model of the aircraft
Capacity	INT	NOT NULL	Total seating capacity
Manufacturer	VARCHAR(50)		Manufacturer of the aircraft
MaintenanceStatus	ENUM('Active', 'In Maintenance')	NOT NULL	Current maintenance status

6. Analytics Table
Stores data for predictive analytics.

Column Name	Data Type	Constraints	Description
AnalyticsID	INT	PRIMARY KEY	Unique identifier for analytics data
FlightID	INT	FOREIGN KEY	References Flights(FlightID)
PassengerCount	INT		Number of passengers on the flight
Revenue	DECIMAL(10, 2)		Total revenue generated by the flight
FuelConsumption	DECIMAL(10, 2)		Fuel consumed during the flight
PredictedDelay	INT		Predicted delay in minutes
